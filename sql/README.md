# 文创手作定制交易平台 - 数据库安装指南

为了简化部署流程，我们已将所有零散的 SQL 补丁整合为 5 个核心文件。请按照编号顺序依次执行。

## 📂 安装包目录：`sql/install/`

### 1. 基础架构安装 (`01_framework_base.sql`)
- **包含内容**: 若义基础表、Quartz 定时任务表、注册功能配置。
- **说明**: 任何环境下的首次部署必须先执行此文件。

### 2. 业务核心安装 (`02_business_core.sql`)
- **包含内容**: 创作者管理、商品分类、定制需求、报价系统、订单与支付系统。
- **说明**: 包含了文创平台的核心交易逻辑表结构。

### 3. 社交与交互安装 (`03_social_interaction.sql`)
- **包含内容**: 作品分享、双轨制评论系统、社区交互（点赞/关注/分享）、实时聊天系统、品牌去标识化补丁。
- **说明**: 包含了所有用户互动与前端展示增强功能。

### 4. 系统菜单与权限 (`04_system_menus.sql`)
- **包含内容**: 后台管理菜单、创作者中心菜单、权限分配初始化。
- **说明**: 执行后请刷新后台界面或重新登录以查看新菜单。

### 5. 演示测试数据 (`05_test_data.sql`)
- **包含内容**: 预置的创作者、商品、分类、作品及模拟评论数据。
- **说明**: **可选执行**。仅建议在开发或演示环境中使用。

---

## ⚠️ 技术备注：关于评论系统
本项目目前采用“双轨制”评论实体映射，`03_social_interaction.sql` 中同时包含了 `creative_comment` 和 `creative_interaction_comment`。Java 层的 `CreativeComment` 实体类已完成合并，可完美兼容这两张表。
